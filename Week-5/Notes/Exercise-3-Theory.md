# Exercise 3 - Functional Components

## Objective
- Understand Functional Components.
- Apply CSS styles.
- Pass data using Props.
- Render a Functional Component.

## Functional Component
A Functional Component is a JavaScript function that returns JSX. It is simpler than a Class Component and is widely used in modern React.

## Props
Props (Properties) are used to pass data from a parent component to a child component.

Example:

<CalculateScore
Name="Tanu"
School="DAV Public School"
total={500}
goal={2}
/>

## CSS in React
CSS files can be imported using:

import "../StyleSheets/mystyle.css";

## JSX
JSX allows HTML-like syntax inside JavaScript.

## Advantages of Functional Components
- Simple syntax
- Better readability
- Easy to maintain
- Supports Hooks
