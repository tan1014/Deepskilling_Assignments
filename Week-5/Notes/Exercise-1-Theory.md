## What is SPA?
A Single Page Application (SPA) loads a single HTML page and dynamically updates the content without reloading the entire page.

### Benefits of SPA
- Faster navigation
- Better user experience
- Reduced server requests
- Smooth page transitions

## What is MPA?
A Multi Page Application (MPA) loads a new HTML page whenever the user navigates to another page.

## SPA vs MPA

SPA:
- Single HTML page
- Fast navigation
- No full page reload

MPA:
- Multiple HTML pages
- Slower navigation
- Reloads each page

## What is React?
React is an open-source JavaScript library developed by Facebook for building user interfaces using reusable components.

## Virtual DOM
Virtual DOM is a lightweight copy of the Real DOM. React compares changes in the Virtual DOM and updates only the necessary parts of the Real DOM, improving performance.

## Features of React
- Component-based architecture
- JSX
- Virtual DOM
- One-way data binding
- Reusable components
- Fast rendering
